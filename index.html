The "Not Messing Around" Quiz

Ideal for: "High-stakes" assessments or those too difficult to make "uncheatable."

Restricts students from:

  • Navigating away from the quiz page
  • Using an external display
  • Using Chrome extensions (outside of assistive technology)
  • Using "Incognito" windows
  • Downloading, printing, or saving quiz content
  • Disabling camera/mic/screen recording
  • Leaving existing tabs open
  • Opening new tabs or links
  • Copying and pasting text to or from the quiz
  • Taking screenshots
  • Leaving any traces of the quiz after submission
  • Using the "right-click" function
  • Re-entering the quiz (w/out Proctorio Support)

Allows students to:

  • Remain solely in the quiz
  • Use one display
  • Rely on knowledge w/ no external resources

The "Make Good Choices" Quiz

Ideal for: Assessments asking students to use specific resources outside of the quiz.

Restricts Students From:

  • Using an external display
  • Using Chrome extensions (outside of assistive technology)
  • Using "Incognito" windows
  • Disabling camera/mic/screen recording
  • Leaving existing tabs open
  • Opening links not provided in the quiz
  • Leaving any traces of the quiz after submission
  • Re-entering the quiz (w/out Proctorio Support)

Allows Students To:

  • Remain (mostly) in the quiz
  • Use one display
  • Open only the links provided in the quiz
  • Use an outside application
  • Rely on knowledge and approved resources

The "Uncheatable" Quiz

Ideal for: All assessments, when possible. OHS may require using Proctorio for "high-stakes" assessments.

Do you even need Proctorio?

This scenario questions whether Proctorio is necessary at all.

Allows Students To:

  • Rely on knowledge with some external resources
  • Have full autonomy to show their learning

Enable These Lock Down Options:

No lock down options are enabled in this scenario.

How to Enable Lock Down Options

  1. Choose the Scenario that best fits your goal for the student experience while taking a Canvas quiz w/ Proctorio.
  2. Turn ON the recommended Lock Down Options (Canvas Quiz > Build > Proctorio Settings > Proctorio Exam Settings > Lock Down Options).
  3. Leave all other options turned OFF (unless otherwise needed).
Note: For Recording Options, turn on Record Video, Record Audio, Record Screen, and Record Web Traffic.
