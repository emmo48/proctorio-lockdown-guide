How to Enable Lock Down Options

1

Choose Your Scenario

Select the scenario that best fits your goal for the student experience while taking a Canvas quiz with Proctorio.

2

Navigate to Settings

Go to Canvas Quiz > Build > Proctorio Settings > Proctorio Exam Settings > Lock Down Options.

3

Configure Recording

For Recording Options, turn on Record Video, Record Audio, Record Screen, and Record Web Traffic.

4

Enable Options

Turn ON the recommended Lock Down Options for your chosen scenario. Leave all other options turned OFF unless otherwise needed.

The "Strict" Quiz

Ideal for: High-stakes assessments with strict parameters.

Restricts students from:

  • Navigating away from the quiz page
  • Using an external display
  • Using Chrome extensions (outside of assistive technology)
  • Using "Incognito" windows
  • Downloading, printing, or saving quiz content
  • Disabling camera/mic/screen recording
  • Leaving existing tabs open
  • Opening new tabs or links
  • Copying and pasting text to or from the quiz
  • Taking screenshots
  • Leaving any traces of the quiz after submission
  • Using the "right-click" function
  • Re-entering the quiz (w/out Proctorio Support)

Allows students to:

  • Remain solely in the quiz
  • Use one display
  • Rely on knowledge w/ no external resources

The "Flexible" Quiz

Ideal for: Assessments allowing for student choice with clear parameters.

Restricts students from:

  • Using an external display
  • Using Chrome extensions (outside of assistive technology)
  • Using "Incognito" windows
  • Disabling camera/mic/screen recording
  • Leaving existing tabs open
  • Opening links not provided in the quiz
  • Leaving any traces of the quiz after submission
  • Re-entering the quiz (w/out Proctorio Support)

Allows students to:

  • Remain (mostly) in the quiz
  • Use one display
  • Open only the links provided in the quiz
  • Use an outside application
  • Rely on knowledge and approved resources
